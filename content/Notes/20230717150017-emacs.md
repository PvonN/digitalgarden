#### Citation and Footnotes {#citation-and-footnotes}

To use proper citation features in Emacs you need to create a
`bibliography.bib` file and create a bibliography item in your Org-File,
where you link this file.

```emacs-lisp
#+BIBLIOGRAPHY: bibliography.bib
```

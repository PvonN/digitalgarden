## Overview {#overview}

Magit is the integrated [Git]({{< relref "20230531074345-git.md" >}})  Package for [Emacs]({{< relref "20230717150017-emacs.md" >}}). You can use the
full Git functionalty from inside Emacs.


## Content {#content}


### Basic Commands {#basic-commands}

-   `C-x g` - open the status buffer
-   `g` - refresh the status buffer
-   `cursor on commit - RET` - show commit details
-   `q` - quit Magit buffer
-   `cursor on commit - d - d` - show dif of commit


### Init Git Project {#init-git-project}

-   in a non-git project dir -&gt; `C-x g`
    -   choose dir to create project from


### Connect to a existing Repository {#connect-to-a-existing-repository}

-   in Magit status buffer (`C-x g`)
    -   `M` - open Remotes buffer
    -   `a` - add Remote
    -   choose name (origin)
    -   add URL from [GitHub]({{< relref "20240131080121-github.md" >}}) or other service
-   now you can Fetch from the Remote Repo
    -   `F` - for Fetch Buffer
-   or Push directly to it
    -   `P` - open Push Buffer


### Adding and Commiting {#adding-and-commiting}

-   call / refresh the status buffer
    -   `C-x g` - open status buffer
    -   `g` - refresh the status buffer
-   stage changes
    -   `cursor on change - s` - stage a single change
    -   `S` - stage all changes
    -   `cursor on staged change - u` - unstage single change
    -   `U` - unstage all changes
-   commit changes
    -   `c c` - commit changes
    -   add description
        -   first line: headline
        -   third line: description
    -   `C-c C-c` - finish commit


### Push Changes to Repository {#push-changes-to-repository}

-   `P` - open push buffer


### Branching {#branching}


#### Create a new branch {#create-a-new-branch}

-   `b` - open branch overview
    -   `n` - for new branch
    -   choose a branch from which you want to create a new branch
    -   give it a name
-   `b l` - checkout new branch


### Merging {#merging}

-   go to branch you want to merge to
    -   `b` - open branch overview
    -   `l` - choose local branch
-   open merge overview `m`
    -   `m` - choose branch to merge from


### Pulling {#pulling}

-   `F` - open pull overview


### Stashing {#stashing}

-   `z` - stash menu


### Checkout {#checkout}


#### Checkout Commit {#checkout-commit}

-   `l l` - go to log menu on current branch
-   set cursor on commit you want to checkout
-   `b b` and choos commit id

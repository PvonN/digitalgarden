## Overview {#overview}

The Bela Board is a optimised microcontroller for sound application. It can be used for extending analog instruments, sound installations or physical controller for music applications.
It runs [Csound]({{< relref "20230718082724-csound.md" >}}), [SuperCollider]({{< relref "20230831211246-supercollider.md" >}}) and a few other sound realted programming environments.


## Content {#content}


### Syncing Files {#syncing-files}

You can use [rsync]({{< relref "20250327105736-rsync.md" >}}) for syncing files from your local machine to your bela board.
Use your command line on your local machine and sync your files like this:

```shell
rsync -av /path/to/your/local/file/or/dir root@bela.local:/root/Bela/
```


### Compiling on Bela Board {#compiling-on-bela-board}

Connect with your board via webbrowser and the URL `bela.local` and use the [Linux]({{< relref "20250327111241-linux.md" >}}) commandline of the Bela Board. You can just use every compile command which runs on Linux like:

```shell
gcc -O2 -I /usr/include/csound -fPIC xine.c -shared -o xine.so
```

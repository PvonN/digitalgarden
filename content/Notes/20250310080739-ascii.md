## Overview {#overview}

ASCII stands for American Standard Code for Information Interchange.
Originally defined as a 7-[Bit]({{< relref "20250307075751-bits.md" >}}) code, it allows the representation of
128 different characters, including letters, numbers, punctuation,
and control characters. Today, an 8-bit code is often used, which
builds on ASCII and enables additional characters by utilizing the
eighth bit. In computer programs, the ASCII code is used to represent
and store these characters.

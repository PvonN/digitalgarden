## Overview {#overview}

CLOS is the "[Common Lisp]({{< relref "20230813193858-lisp.md" >}}) Object System". It allows object
oriented programming in Common Lisp. It was added to the Common Lisp
Language, before the fomalization of the language as an [ANSI]({{< relref "20250214161555-ansi.md" >}}) standard.
Since then it's part of Common Lisp.

You can find here a small summary of the core of it's functionality.
